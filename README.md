# Dual 40mm fan shroud for Monoprice Selet Mini (v2)

__More fans go brrrrr^2!!!__

Replacement part for Monoprice Select Mini (V2) to hold two 40mm fans. This gives an extra fan for part cooling & hotend cooling.
